How it works. The Ask engine is an n-gram rulebook distilled from a local LLM's answers: a question becomes the context … kenya ? a : and the rules complete it. If no rule deep enough to have seen your entity fires, it says I don't know instead of guessing — abstention, not hallucination. The Generate engine is a template automaton mined from schema-constrained LLM text: it picks sentence frames it can fill from your facts (and stops when facts run out), with a tone dial that re-weights frames along a validated urgency direction in embedding space. Every output row cites the rule or template that produced it.
