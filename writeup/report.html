The decomposition is what makes the collapse informative. The polarity axis — one direction in qwen's 1024-dimensional space — reproduced ministral's positive vs. negative call at 0.951 on the unseen set, essentially undiminished. Every bit of the fidelity loss lived in the third class: 17 of 20 errors involved “neutral.”

02“Neutral” was never a region — it's the uncertainty

Sampling ministral seven times per sentence exposed where the target model disagrees with itself, and it lines up with the geometry.

Ministral's self-agreement splits cleanly by class: 0.96 on positive, 0.92 on negative, but only 0.75 on neutral. “Neutral” is where the base model waffles. And distance from the polarity midpoint tracks that waffling directly — Spearman ρ = +0.557 (p = 3×10⁻⁹) between |polarity| and ministral's own agreement. The band you drag above isn't detecting a neutral meaning; it's detecting the model's doubt. That reframes the whole surrogate: not a shaky three-way classifier, but a calibrated binary one that knows when to abstain.

# the entire model, read off one named direction in latent space
polarity(x) = x · unit(μ_positive − μ_negative)   # a single 1024-d axis

if abs(polarity) < m:   abstain     # ← the tunable knob you dragged
elif polarity > 0:      positive
else:                  negative

03What the code revealed about the space

GEOMETRY

Sentiment is one axis, not two

Positive and negative prototypes are more similar to each other (cos 0.808) than either is to neutral — they're both “opinionated,” separated along a single polarity direction.

TRANSFER

The axis generalizes; the hand-tuned gate did not

Polarity held at 0.951 on author-independent text. The earlier “evaluativeness gate” was an artifact of my flat, factual neutral sentences and broke on naturalistic ones.

CALIBRATION

Confidence is legible, not a black box

The surrogate's abstention region catches 85% of the sentences ministral itself is unsure about — the interpretable rule and the neural model doubt the same inputs.

CONTROL

One number is the business dial

The margin m trades coverage for precision along a smooth curve: 74% coverage at 93% precision, 53% at 100%. Pick a floor, route the rest to a human.

Method. ministral-3:3b assigns a one-word sentiment token (the target we replicate); qwen3-embedding:0.6b supplies the 1024-d, L2-normalized input; the surrogate is hand-authored rules over prototype-difference axes, fit only on an original 72-sentence corpus and frozen. The held-out set is 98 sentences written by gemma4:e4b — a different model from both the labeler and the author — labeled by ministral and scored without refitting. Self-consistency uses 7 samples per sentence at temperature 0.8.

Fidelity = agreement with the base model's token, not with human ground truth — the goal is to explain the model, quirks included. Hover any point to read its sentence.
